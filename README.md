# Terarium

## Installation

```sh
git clone https://github.com/undeadpelmen/terarium-compose.git

cd terarium-compose
```

## Usage

1. Front: [http://localhost/](http://localhost/)
2. Back: [http://localhost/api/v1](http://localhost/api/v1)
3. Postgres: [http://localhost:5432](http://localhost:5432)
4. Rabbit: [http://localhost:15672](http://localhost:15672)
